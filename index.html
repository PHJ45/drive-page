We like the command line

Developers spend a lot of time on in the terminal. We make a lot of cool alias's and functions to help us move around the computer.

But what if someone spills coffee on your laptop and you need to use a spare? Or if you need to drive on a different machine while pairing?

We made dis gem called drive

Install the gem. Run it for the first time with the "new" flag (-n). It will copy your bash profile, create a local drive_profile repo and push it up to github.

Now anytime a computer has the gem installed, you can load your bash profile for a given terminal session. Close that terminal window and things are back to normal.
